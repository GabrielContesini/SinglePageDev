Conhecimentos

HTML

A Linguagem de Marcação de Hipertexto (HTML) é uma linguagem de computador que compõe a maior parte das páginas da internet e dos aplicativos online.

Java Script

De modo geral, o JavaScript é uma linguagem que suporta a programação orientada a objetos. Ainda que não possua várias características desta orientação nativamente, é possível realizar algumas emulações.

ReactJs

ReactJs ou React é uma biblioteca JavaScript usada para construir interfaces de usuário, permitindo a elaboração de aplicativos baseados em JavaScript. Os aplicativos React são executados no navegador e não precisam esperar por uma resposta do servidor

NodeJs

O Node.js pode ser definido como um ambiente de execução Javascript server-side.

VueJs

é um framework progressivo para a construção de interfaces de usuário. Ao contrário de outros frameworks monolíticos, Vue foi projetado desde sua concepção para ser adotável incrementalmente.

Angular

De forma simplificada, ele é um Framework. Um frame é uma estrutura de trabalho. Basicamente, é uma ferramenta que permite aos desenvolvedores trabalhar com mais eficiência e de forma mais organizada.
